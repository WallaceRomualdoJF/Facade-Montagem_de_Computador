Implementar esse padrão Facade com o exemplo de Montagem de computador, logo abaixo:

1. Classe Computador: esta classe representa um computador. Ela possui métodos para ligar e desligar o computador, bem como para executar tarefas.

2. Classe CPU: esta classe representa a CPU do computador. Ela possui métodos para ligar e desligar a CPU, bem como para executar tarefas.

3. Classe DiscoRigido: esta classe representa o disco rígido do computador. Ela possui métodos para ligar e desligar o disco rígido.

4. Classe MemoriaRam: esta classe representa a memória RAM do computador. Ela possui métodos para ligar e desligar a memória RAM.

5. Classe MontagemComputadorFacade: esta classe é a Facade que irá fornecer uma interface unificada para a montagem de computadores. Ela usa as outras classes para ligar e desligar componentes do computador e executar tarefas.
